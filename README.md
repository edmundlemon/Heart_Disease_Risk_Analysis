### Description
This is an interactive app that allows user to check their risk of having heart attack. The dataset used is collected by CDC through behavioral questions. The app uses the dataset to train a machine learning model to perform risk prediction for educational purposes.

### To Start The *Streamlit Application*
Run the following command with your environment ready (Anaconda or equivalent) with streamlit installed
```bash
streamlit run app.py
```
After running the command above, it will open up a new browser page for you to use the streamlit app.

### To run the *Data Mining Pipeline*
Make sure the all packages are installed and command primpt is in your environment (Anaconda or equivalent)
Run all cells or execute cell by cell to run the pipeline.
